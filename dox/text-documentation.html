+-----------------------------------------------+
|                  API Overview                 |
+-----------------------------------------------+
| Routes               | HTTP Method | Action   |
+----------------------+-------------+----------+
| /                    | GET         | Info     |
| /                    | POST        | Play     |
+----------------------+-------------+----------+

+---------------------------------------------------+
| POST / JSON Body Structure                        |
+---------------------------------------------------+
| {                                                 |
|     "message": "string",  // Audio message        |
|     "volume": "number",   // Volume level         |
|     "voice": "string"     // Voice type           |
| }                                                 |
+---------------------------------------------------+

+---------------------------------------------------+
| Response Structure (Example for POST /)           |
+---------------------------------------------------+
| {                                                 |
|     "status": "Sound played",                     |
|     "cached": "boolean",  // Cache status         |
|     "time": "number"      // Response time (ms)   |
| }                                                 |
+---------------------------------------------------+
